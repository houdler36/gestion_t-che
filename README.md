## 📋 Description

Application de gestion de projet et de suivi des tâches.

## 🚀 CI/CD

Les tests Django sont lancés automatiquement via **GitHub Actions** à chaque `push` et `pull_request` sur la branche `main`.

## 📊 Couverture de code

Le workflow applique un seuil de couverture minimale (`fail-under=70`).

## 🛠️ Technologies

- Django 4.x
- PostgreSQL
- Bootstrap 5
- GitHub Actions (CI/CD)
